# Layer 7 — Creases Everywhere: Cross-Domain Evidence

**The 90° crease structure appears not only in mathematics and physics,
but across biology, economics, social systems, information theory,
music, and cognition. This layer catalogs the evidence.**

Each document below identifies the same three-part pattern:
- **Fold (J):** the 90° transition
- **Unfold (∫):** the relaxation or integration
- **Crease (Nⱼ ≠ 0):** the locus of non-integrability
